# Utensilios básicos de cocina

## Cuchillos básicos
- **Cuchillo de chef**: versátil, ideal para cortar verduras, carnes y frutas.  
- **Cuchillo de pan**: con sierra, perfecto para rebanar pan sin aplastarlo.  
- **Cuchillo de pelar**: pequeño, ideal para pelar y cortar frutas o verduras pequeñas.

## Ollas y sartenes
- **Olla grande**: para sopas, guisos y hervir alimentos en grandes cantidades.  
- **Olla mediana**: versátil, útil para arroces, pastas y estofados.  
- **Sartén antiadherente**: perfecta para freír y saltear sin que se pegue la comida.  
- **Cacerola pequeña**: ideal para salsas, hervir huevos o calentar pequeñas cantidades.

## Utensilios auxiliares
- **Espátulas**: para remover, mezclar y voltear alimentos.  
- **Batidor de mano**: para mezclar ingredientes líquidos o batir huevos.  
- **Cucharas de madera**: resistentes y seguras para utensilios antiadherentes.  
- **Pinzas de cocina**: para manipular alimentos calientes sin quemarse.

## Recomendaciones de cuidado
- Lavar a mano los cuchillos y secarlos inmediatamente para evitar oxidación.  
- No usar estropajos metálicos en sartenes antiadherentes.  
- Guardar los utensilios en un lugar limpio y seco.
